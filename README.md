# Malmvägen 1, Ädelfors - Multilingual Property Landing Page

A modern, responsive landing page for a historic Swedish "Disponentvilla" (Manager's Villa) in Ädelfors, the "Gold Kingdom" of Småland. Built with pure HTML, CSS, and JavaScript — no frameworks required.

## Features

- **4 Languages**: Swedish, German, Dutch, and English with instant switching
- **Responsive Design**: Mobile-first approach, works on all devices
- **Modern UI**: Scandinavian minimalist aesthetic with Mäklarhuset brand colors
- **Smooth Animations**: Scroll-based fade-ins, parallax hero, counter animations
- **Contact Form**: Ready for backend integration
- **SEO Optimized**: Open Graph tags, semantic HTML5, meta descriptions

## Property Highlights

- **Address**: Malmvägen 1, 57454 Holsbybrunn, Ädelfors
- **Type**: 1.5-plan villa with basement, built 1750
- **Area**: 365 m² living space on a 4,505 m² plot
- **Rooms**: 9 rooms including 7 bedrooms
- **Extras**: 3 rental cottages generating ~150,000 SEK/year
- **Price**: 3,975,000 SEK

## Tech Stack

- HTML5 (semantic)
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript ES6+
- Google Fonts (Inter, Playfair Display)
- Font Awesome 6 Icons

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AdelConsult/maklarhuset-landing.git
   ```

2. Open `index.html` in your browser — no build step required.

3. For development with live reload:
   ```bash
   npx serve .
   ```
